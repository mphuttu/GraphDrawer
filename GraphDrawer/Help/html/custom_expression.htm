Custom Expression y = f(x)

GraphDrawer can plot any mathematical expression of the form y = f(x) where x is the independent variable. Computation runs in a background thread so the user interface stays responsive.

How to Use

  1. In the Draw Functions palette, type your expression in the y = text field.
    Example: x^2 - x + 1
  2. Tick the Draw Custom Function checkbox — or press OK to apply all settings at once.
  3. The graph appears in golden/yellow after a brief computation.
Changing the expression text while the checkbox is ticked immediately restarts the computation and redraws the graph.

Note on domain errors: Points where the expression is undefined (e.g. log(-1), sqrt(-4), division by zero) are simply skipped — the curve will have a gap at those locations.

Plotting Range and Resolution

The custom expression is evaluated for x ∈ [−20, +20] with a step of 0.0005, giving 80 000 sample points. The result is plotted using connected line segments; large discontinuities (asymptotes) automatically lift the pen.
