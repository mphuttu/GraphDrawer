Draw Options

The Draw Options dialog is opened via Draw › Draw Options… or the toolbar button. Changes take effect when you press OK.

ControlDescription
Show Coordinate Axes Toggle the X and Y axes.
Show Ticks Draw evenly-spaced tick marks on both axes.
Ticks Interval Distance between consecutive tick marks, in logical units (100 = 1 cm at screen resolution).
Show Ticks Label Print numeric labels next to each tick mark.
Choose Colour for Coordinate Axes Click the colour button to pick the axis line colour.
Choose Thickness for Coordinate Axes Select line thickness from the drop-down (1 pt to 5 pt).
Background Colour Click the colour button to change the drawing area background.
