User Interface

Main Window

The main window displays the Cartesian coordinate system and all enabled function graphs. It supports scrolling (scroll bars) and print preview.

Toolbar

The toolbar provides quick access to New, Open, Draw Options, Save, Cut, Copy, Paste, Print and About.

Status Bar

The status bar shows the current coordinate axis size (half-widths in logical units) on the right side.

Draw Functions Palette

A floating, always-visible panel containing:

  • A text field for the custom expression (y =).
  • A Draw Custom Function checkbox to enable plotting of the expression.
  • Individual checkboxes for 14 built-in functions.
  • OK — applies all current checkbox settings and starts drawing.
  • Cancel — hides the palette (use View › Draw Functions to reopen).

Checking or unchecking any individual function checkbox immediately updates the graph without needing to press OK.
