Built-in Functions

The following functions can be enabled via the Draw Functions palette. Each is plotted in a distinct colour over the current visible coordinate range.

Draw Cosiney = cos(x) Green
Draw Tangenty = tan(x) Blue Asymptotes at x = ±π/2, ±3π/2, …
Draw Cotangenty = cot(x) Orange-red Asymptotes at x = 0, ±π, …
Draw Exponenty = ex Dark red Grows very rapidly for large x.
Draw Natural Logarithmy = ln(x) Teal Defined only for x > 0.
Draw Arcus Siney = arcsin(x) Magenta Defined for x ∈ [−1, 1], range [−π/2, π/2].
Draw Arcus Cosiney = arccos(x) Yellow-green Defined for x ∈ [−1, 1], range [0, π].
Draw Arcus Tangenty = arctan(x) Orange Range (−π/2, π/2).
Draw Arcus Cotangenty = arccot(x) Yellow
Draw Hyperbolic Siney = sinh(x) Red
Draw Hyperbolic Cosiney = cosh(x) Dark green Always ≥ 1.
Draw Hyperbolic Tangenty = tanh(x) Steel blue Range (−1, 1).
Draw Hyperbolic Cotangenty = coth(x) Purple Asymptote at x = 0.
All built-in functions are plotted over the current visible x-range. Zoom or pan the view, or change the range in Draw Options, to see more or fewer periods.

User-Defined Curves

In addition to the 14 built-in functions you can add your own curves — either y = f(x), parametric x(t), y(t), polar r = f(k), or implicit f(x, y) = 0 — using the Add… button in the User-Defined Curves group.

To edit an existing curve, select it in the list and click Edit…. The Add Curve dialog reopens pre-filled; click OK to apply your changes.

All range fields (x from/to, t from/to, φ from/to) accept constant expressions such as 2*pi, -pi/2, or e^2 in addition to plain numbers.

See Custom Expression & Parametric Curves for details and examples.
