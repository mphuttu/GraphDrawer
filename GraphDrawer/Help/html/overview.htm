GraphDrawer — Overview

GraphDrawer is a Windows MFC application for plotting mathematical functions on a 2-D Cartesian coordinate system. It was written in C++ using the Microsoft Foundation Classes (MFC) and targets both 32-bit (Win32) and 64-bit (x64) Windows platforms.

Features

  • Draw up to 14 built-in functions simultaneously (sine, cosine, tangent, etc.).
  • Enter an arbitrary expression y = f(x) using the custom expression field.
  • Plot parametric curves — enter x(t) and y(t) expressions with a t-range (e.g. a cycloid, a circle, a Lissajous figure).
  • Add and manage multiple user-defined curves — each with its own label, colour, and visibility checkbox.
  • Zoom and pan the coordinate view with the mouse wheel and left-button drag.
  • Set the visible axis range (x min/max and y min/max) in Draw Options.
  • Choose a scale mode: Equal Scale (true 1:1 aspect, circles look circular), Free Scale (custom cm/unit for each axis), or Semi-Log (logarithmic Y axis).
  • Customise coordinate axes: colour, line thickness, tick marks and labels.
  • Choose a background colour.
  • Print and print-preview — output matches the on-screen view exactly.
  • Background thread computation for custom expressions — the UI stays responsive.

Quick Start

  1. Launch GraphDrawer. The Draw Functions palette appears automatically.
  2. Tick one or more built-in function checkboxes — the graph updates instantly.
  3. To plot a custom expression, type it in the y = field (e.g. x^2 - x + 1) and tick Draw Custom Function.
  4. Click Add… in the User-Defined Curves section to add a y = f(x) or parametric curve.
  5. Use Draw › Draw Options to change colours, axis style, and the visible coordinate range.
  6. Scroll the mouse wheel to zoom; drag to pan; right-click to reset the view.
Tip: The Draw Functions palette can be shown/hidden via View › Draw Functions or by closing the palette window.
