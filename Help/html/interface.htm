User Interface

Main Window

The main window displays the Cartesian coordinate system and all enabled function graphs. The visible range, zoom level, and pan position are all controlled interactively with the mouse (see Mouse Controls).

Toolbar

The toolbar provides quick access to New, Open, Draw Options, Save, Cut, Copy, Paste, Print and About.

Status Bar

The status bar shows the current document information on the right side.

Draw Functions Palette

A floating, always-visible panel containing:

  • A text field for the custom expression (y =) and its x-range.
  • A Draw Custom Function checkbox to enable plotting of the expression.
  • Individual checkboxes for 14 built-in functions.
  • A User-Defined Curves group with:
    • A checkbox list of all user-added curves — uncheck to hide, check to show.
    • Add… — opens the Add Curve dialog to add a new y = f(x) or parametric curve.
    • Remove — deletes the selected curve from the list.
  • OK — applies all current settings.
  • Cancel — hides the palette (use View › Draw Functions to reopen).

Add Curve Dialog

Opened via the Add… button. Allows you to define a new curve:

  • y = f(x) — enter an expression in x and optionally restrict the x-range.
  • Parametric — enter separate x(t) and y(t) expressions and the t-range.
  • Choose a label and a colour for the curve.

Geometric Figures

The Geometric Figures dialog (opened via Draw › Geometric Figures…) lets you place labelled geometric objects on the coordinate plane. Objects are listed in the Geometric Objects panel; use Show/Hide to toggle individual object visibility without removing them.

Line Segment

Specify two endpoints (x1, y1) and (x2, y2) and an optional name label. You can also generate:

  • Parallel line — a second segment parallel to the first, offset by a given distance.
  • Transversal line — a line through a specified point at a given angle (degrees).

Triangle

Specify three vertices A, B, C. The triangle dialog offers:

  • Custom labels for each vertex, each side (a, b, c), and each angle (α, β, γ).
  • Triangle type: General, Right-angled (angle C = 90°), Isosceles (AC = BC), or Equilateral.
  • Show options:
    • Vertex labels, side labels, angle labels, angle values in degrees.
    • Circumcircle — circumscribed circle (cyan outline).
    • Incircle — inscribed circle (green outline).
    • Perp. bisectors — perpendicular bisectors of each side (dashed purple).
    • Angle bisectors — bisectors from each vertex to the opposite side (dashed orange).
    • Altitudes — perpendiculars from each vertex to the opposite side (solid pink/magenta).
    • Medians — lines from each vertex to the midpoint of the opposite side (solid blue).
