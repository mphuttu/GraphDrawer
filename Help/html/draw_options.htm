Draw Options

The Draw Options dialog is opened via Draw › Draw Options… or the toolbar button. Changes take effect when you press OK.

Axis Style

ControlDescription
Show Coordinate Axes Toggle the X and Y axes.
Show Ticks Draw evenly-spaced tick marks on both axes.
Ticks Interval Approximate number of tick subdivisions along each axis. The actual step is rounded to a "nice" number (1, 2, 5, 10, …).
Show Ticks Label Print numeric labels next to each tick mark showing the math value.
Choose Colour for Coordinate Axes Click the colour button to pick the axis line colour.
Choose Thickness for Coordinate Axes Select line thickness from the drop-down (1 pt to 5 pt).
Background Colour Click the colour button to change the drawing area background.

Coordinate Range

These fields control the mathematical extent of the coordinate system shown in the view. After pressing OK the zoom/pan state is reset to this range.

ControlDescription
X minLeft edge of the visible x-axis (default −10).
X maxRight edge of the visible x-axis (default +10).
Y minBottom edge of the visible y-axis (default −10).
Y maxTop edge of the visible y-axis (default +10).

Scale Mode

The Scale Mode radio buttons control how mathematical units are mapped to screen pixels.

ModeDescription
Equal Scale One mathematical unit occupies exactly the same number of pixels on both axes regardless of the monitor's DPI or aspect ratio. A circle defined as x² + y² = 1 is drawn as a perfect circle, not an ellipse. The shorter axis range is expanded automatically to fill the window.
Free Scale Set independent scales for X and Y in centimetres per unit. The Scale X and Scale Y fields become active.
Logarithmic One or both axes are drawn on a base-10 logarithmic scale. Select the desired combination with the sub-mode radio buttons that appear below the Logarithmic scale option (see table below). All logarithmic axis values must be strictly positive; any range boundary ≤ 0 is automatically clamped to 10−3.

Logarithmic Sub-Modes

These radio buttons are enabled only when Logarithmic scale is selected.

Sub-ModeDescription
Log X, Linear Y X-axis is logarithmic, Y-axis is linear. Useful for plotting functions that vary slowly over many orders of magnitude in X (e.g. frequency response).
Linear X, Log Y X-axis is linear, Y-axis is logarithmic (classic semi-log plot). Useful for exponential growth or decay curves.
Log X, Log Y Both axes are logarithmic (log-log plot). Useful for power-law relationships such as y = x^n, which appear as straight lines on a log-log scale.
ControlDescription
Scale X (cm/unit) Centimetres per mathematical unit on the X-axis. Active only in Free Scale mode.
Scale Y (cm/unit) Centimetres per mathematical unit on the Y-axis. Active only in Free Scale mode.
Tip: You can also zoom and pan interactively with the mouse without opening this dialog — see Mouse Controls. Right-click in the view to reset back to the range set here.
