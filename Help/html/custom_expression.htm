Custom Expression y = f(x) and Parametric Curves

Quick Custom Expression y = f(x)

GraphDrawer can plot any mathematical expression of the form y = f(x) where x is the independent variable. Computation runs in a background thread so the user interface stays responsive.

How to Use

  1. In the Draw Functions palette, type your expression in the y = text field.
    Example: x^2 - x + 1
  2. Tick the Draw Custom Function checkbox — or press OK to apply all settings at once.
  3. The graph appears in golden/yellow after a brief computation.
Changing the expression text while the checkbox is ticked immediately restarts the computation and redraws the graph.

User-Defined Curves (Add… / Edit… Dialog)

You can add any number of curves via the Add… button in the User-Defined Curves section of the Draw Functions palette. Each curve has its own label, colour, and visibility checkbox.

y = f(x) Mode

  1. Click Add… and select the y = f(x) radio button.
  2. Enter the expression in x, the x-range, an optional label, and a colour.
  3. Range fields accept constant expressions — you can type 2*pi, -pi, e^2, or any arithmetic combination involving numbers, pi, and e.
  4. Press OK — the curve is computed and added to the list.

Parametric Mode

  1. Click Add… and select the Parametric radio button.
  2. Enter separate x(t) and y(t) expressions using t as the parameter variable.
  3. Set the t-range — you can write 2*pi or 6*pi directly in the range fields.
  4. Press OK.

Polar Mode r = f(k)

  1. Click Add… and select the Polar r = f(t) radio button.
  2. Enter the radius expression in k (the angle variable), and the angle range (e.g. 0 … 2*pi).
  3. Press OK.

Implicit Mode f(x, y) = 0

  1. Click Add… and select the Implicit function radio button.
  2. Enter an expression in x and y whose zero set is the curve.
  3. Set the x and y ranges, then press OK.

Editing an Existing Curve

Select a curve in the list and click Edit… to re-open the dialog pre-filled with the curve’s current settings. Change any field and click OK to replace the curve in-place; click Cancel to leave it unchanged.
